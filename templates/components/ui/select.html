{% comment %} UI-lib компонент: одиночный select без поиска (без внешних UI-библиотек). Параметры include: - select_id (обязательный): id элемента select - select_name (опционально): name элемента select - select_class (опционально): CSS-классы нативного select - root_class (опционально): CSS-классы корневого контейнера - disabled (опционально): True/False, по умолчанию True - placeholder (опционально): текст плейсхолдера компонента - empty_text (опционально): текст пустого списка - loading_option_value (опционально): value стартовой option - loading_option_label (опционально): текст стартовой option - options (опционально): список option-словарей {value, label, disabled, selected} - dropdown_z_class (опционально): z-index класс dropdown (например: z-20, z-[102]) {% endcomment %}
