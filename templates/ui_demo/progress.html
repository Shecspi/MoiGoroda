Прогресс-бары

Примитив: только CSS-классы из frontend/ui-lib/styles/progress.css, без JavaScript.

Все прогресс-бары используют правильные ARIA-атрибуты и поддерживают тёмную тему. Значения прогресса устанавливаются через inline-стили (style="width: X%" или style="height: X%").

Разные размеры

Используйте классы progress-xs, progress-sm, progress-md, progress-lg для управления высотой.

XS (h-1.5)

25% завершено

SM (h-2, default)

50% завершено

MD (h-4)

75% завершено

LG (h-6)

100% завершено
Показать код 
<!-- XS -->
<div class="progress progress-xs" role="progressbar" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100">
  <div class="progress-bar progress-bar-primary" style="width: 25%">
    <span class="sr-only">25% завершено</span>
  </div>
</div>

<!-- SM (default) -->
<div class="progress progress-sm" role="progressbar" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100">
  <div class="progress-bar progress-bar-primary" style="width: 50%">
    <span class="sr-only">50% завершено</span>
  </div>
</div>

Плавающий лейбл

Лейбл располагается над баром и позиционируется с помощью ms-[calc(X%-20px)], где X - процент прогресса.

25%
25% завершено
50%
50% завершено
75%
75% завершено
100%
100% завершено
Показать код 
<div>
  <span class="progress-floating-label ms-[calc(75%-20px)]">75%</span>
  <div class="progress progress-md" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100">
    <div class="progress-bar progress-bar-primary" style="width: 75%">
      <span class="sr-only">75% завершено</span>
    </div>
  </div>
</div>

4 сегмента, 100% - success

Круглые (circular)

Используйте SVG с progress-circular. Значение устанавливается через stroke-dashoffset: для 75% используйте offset=25 (100-75).

75% без текста

75% завершено

35% с текстом

35%
Показать код 
<!-- 75% с текстом в центре -->
<div class="progress-circular progress-circular-lg" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100">
  <svg viewBox="0 0 36 36" xmlns="http://www.w3.org/2000/svg">
    <circle cx="18" cy="18" r="16" fill="none" class="progress-circular-bg" stroke-width="2"></circle>
    <circle cx="18" cy="18" r="16" fill="none" class="progress-circular-bar" stroke-width="2"
            stroke-dasharray="100" stroke-dashoffset="25" stroke-linecap="round"></circle>
  </svg>
  <div class="progress-circular-text">75%</div>
</div>

<!-- Расчёт stroke-dashoffset:
     Для 75%: offset = 100 - 75 = 25
     Для 50%: offset = 100 - 50 = 50
     Для 35%: offset = 100 - 35 = 65 -->

<!-- Расчёт stroke-dasharray для Gauge:
     135° (75% от круга): dasharray="75 100"
