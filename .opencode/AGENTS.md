# Правила проекта

## Лицензия в файлах

- Во все создаваемые и редактируемые файлы нужно добавлять информацию о лицензии:

```text
# ---------------------------------------------
#
# Copyright © Egor Vavilov (Shecspi)
# Licensed under the Apache License, Version 2.0
#
# ----------------------------------------------
```

- Лицензионный блок должен быть обычным комментарием файла, а не docstring/module docstring.
- В Python-файлах размещайте лицензионный блок перед module docstring и импортами.

# Правила работы с сервером

- Заходить на серверы (moi-goroda.ru и любые другие) запрещено.
- Если пользователь просит разобраться в чём-то после `check-server` — не предпринимать действий на сервере, а дать пользователю пошаговую инструкцию, что выполнить в консоли, и попросить показать вывод.
- Вся диагностика сервера — только через переданные пользователем выводы команд.
