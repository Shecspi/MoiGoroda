mg-select / mg-select-searchable (ui-lib)

mg-select — вариант без поля поиска (регистрация: data-component="mg-select").

mg-select-searchable — вариант с фильтрацией (регистрация: data-component="mg-select-searchable").

В Django-шаблонах используйте include: components/ui/select.html и components/ui/select_search.html.

mg-combobox-static / mg-combobox-remote (ui-lib)

Компоненты без React/Vue: шаблон + data-mg-combobox-* + data-mg-part + автoинициализация ui_core.js по data-component="mg-combobox-static" или mg-combobox-remote.

Два include: combobox_static.html и combobox_remote.html.

Базовый include (статические опции)

{% templatetag openblock %} include "components/ui/combobox_static.html" with
combobox_id="city-search"
input_id="city-search-input"
hidden_name="city_id"
label="Город"
placeholder="Начните вводить..."
options=demo_combobox_options
min_chars="0"
max_items="20"
open_on_focus="true"
autoselect_first="true"
{% templatetag closeblock %}

Основные параметры include

  • combobox_id, input_id, hidden_name
  • options (list dict: value, label)
  • label, placeholder, value, input_value
  • show_leading_icon, root_class, input_class, listbox_class

Поведение (в HTML — data-mg-combobox-…)

  • open_on_focus → data-mg-combobox-open-on-focus
  • min_chars → data-mg-combobox-min-chars
  • max_items → data-mg-combobox-max-items
  • autoselect_first → data-mg-combobox-autoselect-first

Remote-режим (через include combobox_remote.html)

  • source_url — endpoint для поиска (например, /api/city/search).
  • query_param — имя query-параметра (по умолчанию query).
  • country_url_param — дополнительный фильтр из текущего URL.
  • value_key, label_key — поля из JSON-ответа.
  • meta_keys — дополнительные поля второй строкой в опции.
  • debounce, loading_text, empty_text.

События компонента

  • mg:combobox:open — список открыт.
  • mg:combobox:close — список закрыт.
  • mg:combobox:highlight — смена активной опции (detail: value, label).
  • mg:combobox:select — выбор значения (detail: value, label, source).
const node = document.getElementById('city-search-combobox');
node.addEventListener('mg:combobox:select', (event) => {
  const { value } = event.detail;
  window.location.href = `/city/${value}`;
});

Клавиатура и доступность

  • ArrowDown/ArrowUp — перемещение по опциям.
  • Enter — выбор активной опции.
  • Escape или клик вне компонента — закрытие.
  • ARIA: role="combobox", role="listbox", role="option", aria-expanded, aria-activedescendant.

Переинициализация для динамического DOM

// После вставки HTML с data-component="mg-combobox-static" или "mg-combobox-remote"
window.MGUi.initAll(containerElement);

// При удалении фрагмента (чтобы снять обработчики)
window.MGUi.destroyAll(containerElement);
